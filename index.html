TEC WEB

HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear la estructura de contenido en la web. Se utiliza para definir elementos como encabezados, párrafos, enlaces y más.

CSS

CSS (Cascading Style Sheets) se utiliza para controlar el diseño y la apariencia visual de los elementos HTML. Permite aplicar colores, fuentes, márgenes y más a una página web.

JS

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.
